```
binmgr installs binaries from various places and can keep them updated

Usage:
  binmgr [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command
  install     Installs a binary
  list        Lists all installed binaries
  status      Checks for updates and prints if binaries need updating
  uninstall   Uninstalls a binary
  update      Updates all installed binaries

Flags:
  -h, --help              help for binmgr
      --loglevel string   Log level (default "warn")

Use "binmgr [command] --help" for more information about a command.
```
